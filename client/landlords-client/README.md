# doudizhu-stand-alone

[cocos官方主页](https://www.cocos.com/)
=========================


页面

    注册/登录
    首页
    房间列表
    游戏
    
功能
    
    注册、登录、退出、用户信息、房间选择、游戏中数据实时同步、断线重连、游戏

游戏顺序
=========================
注册 -> 登录 -> 大厅 -> 房间 -> 游戏
